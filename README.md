# ChatGpt like chat application

# How to run?
### STEPS:

Clone the repository

```bash
Project repo: https://github.com/
```

### STEP 01- initialize yarn

```bash
yarn init
```

### STEP 02- install the requirements
```bash
yarn install
```

### STEP 03- initialize typescript
```bash
npx tsc --init
```

### Issue with OpenAI
```bash
yarn upgrade openai
```


### Tech-stack Used:

- NodeJS
- ExpressJS
- TypeScript
- OpenAI
